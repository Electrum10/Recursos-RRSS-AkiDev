Tutorial de Animaciones con Keyframes en CSS

¿Qué es una animación con Keyframes?

Las animaciones con @keyframes en CSS te permiten animar los elementos HTML, definiendo cómo cambian sus estilos en diferentes puntos del tiempo. Puedes especificar estos cambios en una serie de pasos o "frames", desde el inicio (0%) hasta el final (100%) de la animación.

Ejemplo 1: Mover una caja

En este ejemplo, una caja se moverá de izquierda a derecha usando @keyframes:

Código CSS:


.box {
    width: 100px;
    height: 100px;
    background-color: #007bff;
    margin: 20px auto;
    animation: move 2s infinite alternate;
}

@keyframes move {
    0% {
        transform: translateX(0);
    }
    100% {
        transform: translateX(200px);
    }
}

Ejemplo 2: Rotar una caja

Este ejemplo muestra cómo hacer que una caja gire continuamente usando @keyframes:

Código CSS:


.rotate-box {
    width: 100px;
    height: 100px;
    background-color: #28a745;
    margin: 20px auto;
    animation: rotate 4s infinite linear;
}

@keyframes rotate {
    0% {
        transform: rotate(0deg);
    }
    100% {
        transform: rotate(360deg);
    }
}
